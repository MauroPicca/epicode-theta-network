# BW1 - Team 3 - Theta Network

## 📌 Descrizione
Progetto di progettazione e simulazione di un'infrastruttura di rete per la Theta Company, con particolare attenzione alla sicurezza, segmentazione e gestione degli accessi.

## 🎯 Obiettivo
Realizzare una rete aziendale sicura ed efficiente, simulata tramite Cisco Packet Tracer e documentata con diagrammi e relazione tecnica.

## 📂 Contenuto del repository
- Topologia di rete (Packet Tracer)  
- Diagramma architetturale (.png)  
- Documento di progetto  
- Relazione tecnica Datashield  

## 🛠️ Tecnologie utilizzate
- Cisco Packet Tracer  
- Network Design  
- Sicurezza di rete
